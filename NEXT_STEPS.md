# Next Steps

如果你准备开始真正冲 star，先做这几件事：

## 1. 先发第一条中文主贴

直接用：

- `docs/final-posts.md`

里的 **中文主贴（推荐最先发）**

## 2. 开启 GitHub Discussions

然后用：

- `docs/discussion-template.md`

里的欢迎贴模板发第一条 discussion。

## 3. 发英文短帖

还是用：

- `docs/final-posts.md`

里的 X / Twitter 版本。

## 4. 观察 24 小时数据

用：

- `docs/metrics-tracker.md`

记录：

- stars
- traffic
- 常见问题
- 反馈点

## 5. 决定第一轮迭代

优先级建议：

1. 安装问题
2. 首页理解成本
3. 截图 / GIF 不清楚
4. `digest` 或飞书桥接理解成本
