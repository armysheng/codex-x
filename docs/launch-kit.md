# Launch Kit

这份文档用来支持 `codex-x` 的首发和后续传播。更完整的增长节奏见 [1k Star Plan](./1k-star-plan.md)。

## 30 秒短介绍

> `codex-x` is a memory-first local workspace toolkit for Codex. It gives Codex durable project context, installs in one command, and includes a plugin unlock skill plus a mobile access skill.

## 中文短介绍

> `codex-x` 是一个给 Codex 用的 memory-first 本地工作区工具箱。它解决几件事：让 Codex 有长期记忆、让工作区一条命令装起来、让飞书桥接、解锁插件 skill 和移动端访问 skill 按需接进本地工作流。

## 首发文案（长版）

可以围绕这 3 个点展开：

1. AI 会话上下文很容易丢
2. 很多人并不缺“更大的平台”，而是缺一个真的能用起来的工作区
3. `codex-x` 先把记忆系统、初始化器、每日记忆整理和可选入口这几个点做好

## 首发文案（中文）

可以直接作为第一版帖子基础：

> 我把自己平时给 Codex 用的工作区骨架整理成了一个开源项目：`codex-x`。
>
> 它不是另一个“大而全 AI 平台”，而是先把 4 件事做好：
> 1. 给 Codex 一个能长期记住上下文的工作区
> 2. 给用户一条命令就能装起来的初始化器
> 3. 用 Codex automation 定时整理 daily memory
> 4. 给需要的人解锁插件 skill 和移动端访问 skill
>
> 现在 README 里已经能看到：
> - 一键安装命令
> - 可直接丢给 Codex 的安装提示词
> - 记忆系统目录结构
> - prompt-native daily memory automation
> - 解锁插件 skill 和移动端访问 skill
>
> 仓库：`armysheng/codex-x`
