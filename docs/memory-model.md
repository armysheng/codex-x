# Memory Model

`codex-x` 采用文件化记忆模型：

- `status.md`：短期快照
- `context.md`：中期上下文
- `memory/YYYY-MM-DD.md`：每日原始记录
- `about-me/MEMORY.md`：长期稳定偏好与结论

## 什么时候写到哪里

### `status.md`

适合写：

- 现在正在做什么
- 当前阻塞
- 今天最重要的临时提醒

不适合写：

- 长期偏好
- 一整周的背景

### `context.md`

适合写：

- 这个阶段在推进什么
- 为什么在做这件事
- 这周的主线和关键变化

### `memory/YYYY-MM-DD.md`

适合写：

- 当天实际发生了什么
- 做了哪些动作
- 学到什么结论

### `about-me/MEMORY.md`

适合写：

- 稳定偏好
- 已验证的长期判断
- 未来仍然值得沿用的规则

## 推荐沉淀路径

优先顺序是：

`当天事实 -> daily memory -> context/status -> about-me/MEMORY`

不要一上来把所有东西都塞进长期记忆。
