# Launch Kit

这份文档用来支持 `codex-x` 的首发和后续传播。

## 30 秒短介绍

> `codex-x` is a memory-first local workspace toolkit for Codex. It gives Codex a real workspace, installs in one command, and can optionally bridge Feishu into your local workflow.

## 中文短介绍

> `codex-x` 是一个给 Codex 用的本地工作区工具箱。它解决三件事：让 Codex 有记忆、让工作区一条命令装起来、让飞书能接进本地工作流。

## 首发文案（长版）

可以围绕这 3 个点展开：

1. AI 会话上下文很容易丢
2. 很多人并不缺“更大的平台”，而是缺一个真的能用起来的工作区
3. `codex-x` 先把记忆系统、初始化器、消息桥接这 3 个最关键的点做好

## 首发文案（中文）

可以直接作为第一版帖子基础：

> 我把自己平时给 Codex 用的工作区骨架整理成了一个开源项目：`codex-x`。  
>  
> 它不是另一个“大而全 AI 平台”，而是先把 3 件事做好：  
> 1. 给 Codex 一个能长期记住上下文的工作区  
> 2. 给用户一条命令就能装起来的初始化器  
> 3. 给需要的人一个可选的飞书桥接入口  
>  
> 现在已经支持：  
> - memory-first workspace template  
> - 一键安装脚本  
> - `doctor / smoke / start / stop / status / logs / send`  
>  
> 仓库：`armysheng/codex-x`
